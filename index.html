How It Works

Start a private, encrypted session in under 30 seconds.

1. Open Video Chat

Grant camera and microphone permissions. Your Room ID is generated locally — no account needed.

2. Share Your Room ID

Send your unique Room ID to your participant via any channel. Only people with the ID can connect.

3. Record Consent

Before connecting, SafeCloak prompts all participants for cryptographically-timestamped consent.

4. Communicate Securely

Your call is WebRTC peer-to-peer. Take encrypted notes and manage voice settings in real time.

Security Architecture

Built on open standards and browser security primitives.

  • ✓ AES-256-GCM encryption via Web Crypto API
  • ✓ PBKDF2 key derivation (100,000 iterations, SHA-256)
  • ✓ WebRTC DTLS/SRTP transport encryption
  • ✓ Peer-to-peer — no media relayed through servers
  • ✓ Notes encrypted at rest in localStorage
  • ✓ SHA-256 consent integrity hashes
  • ✓ No third-party analytics or tracking
  • ✓ Open-source under OWASP BLT
  • ✓ Content Security Policy headers
  • ✓ Browser-only — no backend, no accounts
  • ✓ GDPR-ready consent export (JSON/CSV)
  • ✓ Voice noise suppression & echo cancellation
